Corsair Lighting Effects Documentation

Overview

Corsair Lighting Effect Libraries contain effects that are currently available in CUE as well as some additional effects and a framework for endÂ­users to implement their own effects.

Libraries functional features

  1. CUE LFX: contains definitions of lighting effects available in CUE (wave, ripple, rainbow, rain and etc);
  2. Corsair LFX: contains custom gradient effect and progress bar implementation;
  3. Corsair Layers: acquires frames from every of lighting effects, mixes them into single frame depending on mixing policy and outputs the result to CUE SDK.

Dependencies

Corsair Lighting Effect Libraries depend on CUE SDK and require CUE running with "Enabled SDK" in settings.
